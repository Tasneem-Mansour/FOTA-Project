For STM32 Microcontroller

The Over-The-Air Programmer (OTA) project enables wireless flashing of firmware onto an Arm-based STM32F401 microcontroller via a website linked to Firebase. A NodeMCU board fetches code from the internet and sends it to the STM32,where it's written to Flash Memory using a Bootloader in the first 16 kB of memory.
